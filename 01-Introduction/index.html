01-Introduction

The css file for this page was created from a scss file

using the command,

sass --watch 01-Introduction/scss/style.scss:01-Introduction/css/style.css

  Prerequisite:

  Install gem

  Check if installed using gem -v

  Install sass using gem

  gem install sass

  Check if installed using sass -v

  To convert scss file to css

  sass --watch filename.scss:filename.css
