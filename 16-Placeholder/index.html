16 - Placeholder.







Check the difference having a mixin and a placeholder

SCSS file

%square {
  width: 100px;
  height: 300px;
}

.red {
  @extend %square;
  background: red;
}

.blue {
  @extend %square;
  background: blue;
}

@mixin square {
  width: 200px;
  height: 200px;
}

.mred {
  @include square();
  background: red;
}

.mblue {
  @include square();
  background: blue;
}

Generated CSS file

.red, .blue {
  width: 100px;
  height: 300px; }

.red {
  background: red; }

.blue {
  background: blue; }

.mred {
  width: 200px;
  height: 200px;
  background: red; }

.mblue {
  width: 200px;
  height: 200px;
  background: blue; }
