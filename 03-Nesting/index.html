03-Nesting

The css file for this page was created from a scss file

Even arithmetic operations are possible on sass variables.

Content in main.
Content in sub.
main continuing. 

    $mygreen:#09d22d;
    $myred: #8c1339;

    .main {
      color: $mygreen;
      background-color: $myred;
      .sub {
        color: $mygreen + 100;
        background-color: $myred + 100;
      }
    }
